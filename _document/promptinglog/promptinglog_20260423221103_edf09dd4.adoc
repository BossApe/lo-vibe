リポジトリ名の変更には2つの側面があります。

---

**① GitHub上のリポジトリ名を変更する（推奨：先にこちら）**

1. https://github.com/BossApe/mmmm/settings を開く
2. **Repository name** 欄で新しい名前を入力
3. **Rename** ボタンをクリック

GitHubが自動でリダイレクトを設定しますが、ローカルのremote URLも更新が必要です：

```bash
cd /Users/m.nohara/gitspace/mmmm
git remote set-url origin https://github.com/BossApe/新しい名前.git
```

---

**② ローカルのディレクトリ名を変更する**

```bash
mv /Users/m.nohara/gitspace/mmmm /Users/m.nohara/gitspace/新しい名前
```

`.git`フォルダはそのまま引き継がれるので、git履歴は保持されます。

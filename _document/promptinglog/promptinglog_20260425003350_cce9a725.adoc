作成したファイルと動作概要です。

**作成物**

| ファイル | 説明 |
|---|---|
| [tools/save_promptinglog/main.go](lo-vibe/tools/save_promptinglog/main.go) | メインプログラム |
| [tools/save_promptinglog/go.mod](lo-vibe/tools/save_promptinglog/go.mod) | Goモジュール定義 |
| [tools/save_promptinglog/README.adoc](lo-vibe/tools/save_promptinglog/README.adoc) | ツール説明 |
| [_document/promptinglog/](lo-vibe/_document/promptinglog/) | 出力先ディレクトリ |

**使い方**

```bash
cd tools/save_promptinglog

# ビルド
go build -o save_promptinglog .

# 実行（最新セッションを自動検出）
./save_promptinglog

# セッションIDを指定する場合
./save_promptinglog -session-id <セッションID>

# 出力先を変更する場合
./save_promptinglog -output-dir /path/to/dir
```

**動作内容**

1. `~/Library/Application Support/Code/User/workspaceStorage/**/GitHub.copilot-chat/transcripts/` を再帰検索
2. 最も更新日時が新しい `.jsonl` ファイルを自動選択（`-session-id` 指定時はそのセッション）
3. `user.message` / `assistant.message` をターン単位でパース
4. AsciiDoc（セッション情報テーブル＋ターン別会話ログ）を生成
5. `_document/promptinglog/promptinglog_<yyyymmddhhmmss>.adoc` として保存

## 1. 対応番号

- 3章/4章の対応番号: 9

## 2. 主な機能

- Git リポジトリ管理
- Issue と PR 管理
- チーム権限管理
- Webhook による自動連携

## 3. 運用想定

- 実行場所: Linux サーバの app ネットワーク
- 接続元: 開発者端末、aider
- 接続先: MinIO、将来 CI サービス
- 保全: 定期バックアップとリストア手順整備

- バックアップの復元訓練を定期的に行う

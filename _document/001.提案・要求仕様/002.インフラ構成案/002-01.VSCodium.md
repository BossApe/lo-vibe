## 1. 対応番号

- 3章/4章の対応番号: 1

## 2. 主な機能

- AI コーディング系拡張を統合した開発体験
- Git 操作、差分確認、レビュー作業
- Remote SSH による Linux サーバ上ワークスペースの編集
- タスク実行、デバッグ、テスト実行の統合

## 3. 運用想定

- 実行場所: 開発者端末
- 接続先: Linux サーバ上の開発コンテナ、または Git リポジトリ
- データ保持: ローカル設定、拡張設定、SSH 鍵
- 可用性: 端末障害時は設定を dotfiles で復元

## 5. 運用ルール

- 端末に秘密情報を長期保存しない
- Linux サーバ側でビルドと検証を実行する
- 主要拡張はバージョンを固定して再現性を確保する

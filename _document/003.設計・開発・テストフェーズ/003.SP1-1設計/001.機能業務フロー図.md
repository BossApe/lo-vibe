## 2. FR-001 詳細フロー（SCR-001 / POST /api/v1/system-overviews）

> 📱 **画面**: SCR-001 システム概要入力（`/overview`）  
> 🔌 **API**: `POST /api/v1/system-overviews`、`GET /api/v1/system-overviews/{id}`

```mermaid
sequenceDiagram
    participant User as ユーザ（ブラウザ）
    participant UI as 📱 SCR-001<br>Musuhi フロントエンド<br>(SvelteKit / /overview)
    participant API as 🔌 Musuhi API<br>(Go / :8080)
    participant DB as PostgreSQL

    User->>UI: システム概要テキストを入力・送信
    UI->>UI: フロントエンドバリデーション<br>（空文字・4096字超チェック）
    UI->>API: POST /api/v1/system-overviews<br>{"content": "..."}
    API->>API: サーバサイドバリデーション
    alt バリデーションNG
        API-->>UI: 422 Unprocessable Entity<br>{"error": {"code":"VALIDATION_ERROR",...}}
        UI-->>User: エラーメッセージ表示
    else バリデーションOK
        API->>DB: INSERT INTO system_overviews<br>(content)
        DB-->>API: id, content, created_at
        API-->>UI: 201 Created<br>{"data": {"id":"...","content":"...","createdAt":"..."}}
        UI-->>User: 保存完了・次ステップへ誘導（SCR-002 へ）
    end
```

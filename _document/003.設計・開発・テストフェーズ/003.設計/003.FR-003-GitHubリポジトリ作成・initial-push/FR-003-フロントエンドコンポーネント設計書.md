## 1. 対象ルート

- URL: `/projects/{id}/repository`
- 画面ID: SCR-003
- 目的: GitHub リポジトリ作成と initial push の実行・結果確認

## 2. 状態変数

| 変数 | 型 | 説明 |
| --- | --- | --- |
| owner | string | GitHub owner 入力値 |
| repoName | string | リポジトリ名 |
| visibility | 'public'\|'private' | 公開設定 |
| localPath | string | push 対象パス |
| commitMessage | string | 初回コミットメッセージ |
| repositoryUrl | string | 成功時の URL |
| externalProjectId | string | 成功時のリポジトリ ID |
| pushStatus | string | pending/success/failed |
| isLoading | boolean | 実行中状態 |
| errorMessage | string | エラー表示用 |

## 3. 主要関数

| 関数 | 役割 |
| --- | --- |
| postWithExternal | `POST /api/v1/projects/with-external` 呼び出し |
| handleSubmit | 入力検証後に API 実行し状態を更新 |
| handleRetry | 失敗時に再実行 |

## 4. UI 構成

- 入力フォーム（owner/repoName/visibility/localPath/commitMessage）
- 実行ボタン
- 成功時カード（repositoryUrl, pushStatus）
- 失敗時エラーメッセージ（`role=alert`）

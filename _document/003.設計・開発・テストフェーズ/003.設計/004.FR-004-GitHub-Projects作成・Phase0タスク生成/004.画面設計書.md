## 1. 対象画面

- SCR-004 GitHub Projects確認

## 2. 入力項目

### 2.1 Projects 作成

| 項目 | 必須 | 説明 |
| --- | --- | --- |
| owner | 必須 | GitHub owner |
| title | 必須 | Projects タイトル |

### 2.2 Phase0 タスク作成

| 項目 | 必須 | 説明 |
| --- | --- | --- |
| owner | 必須 | GitHub owner |
| projectsId | 必須 | Projects Node ID |

## 3. 表示項目

- Projects URL
- Projects ID
- 追加済みタスク一覧（id/title/type）
- 実行ステータス（success/failed）

## 4. バリデーション

- owner 未入力時は API 呼び出し不可
- title 未入力時は Projects 作成不可
- projectsId 未入力時は Phase0 タスク作成不可

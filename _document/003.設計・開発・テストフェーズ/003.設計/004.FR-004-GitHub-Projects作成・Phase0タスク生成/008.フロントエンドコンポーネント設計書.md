# 008.フロントエンドコンポーネント設計書

前: [007.エラーハンドリング設計書](007.エラーハンドリング設計書.md) | [一覧](README.md) | 次: [009.シーケンス図（エラー詳細）](009.シーケンス図（エラー詳細）.md)

## 1. 対象コンポーネント

- SCR-004 の GitHub Projects 作成セクション
- SCR-004 の Phase0 タスク作成セクション

## 2. 状態管理

- owner: string
- title: string
- projectsId: string
- projectsUrl: string
- tasks: Array<{id,title,type}>
- isSubmitting: boolean
- errorMessage: string

## 3. 主要操作

- onCreateProjects: POST /api/v1/projects/{id}/github-projects
- onCreatePhase0Tasks: POST /api/v1/projects/{id}/phase0-tasks

## 4. 表示制御

- API 実行中はボタンを disabled
- 成功時に URL と tasks を表示
- 失敗時にエラーメッセージを表示

# FR-002 画面設計書（SCR-002）

前: [002.API仕様書](002.API仕様書.md) | [一覧](README.md) | 次: なし

## 1. 画面概要

- 画面ID: SCR-002
- 画面名: プロジェクト名・構成要素確認
- URL: `/projects/setup`
- 目的: 機能抽出結果確認、プロジェクト名確定、初期ディレクトリ作成

## 2. 入力項目

| 項目 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| 概要ID | 文字列(UUID) | 必須 | FR-001 保存結果ID。URL クエリパラメータ `overviewId` から取得し、画面には表示しない |
| プロジェクト名 | 文字列 | 必須 | 候補選択または手入力。パターン: `^[a-zA-Z0-9][a-zA-Z0-9_-]*$` |
| プロジェクト名由来説明 | 文字列 | 条件付き | AI提案候補を選択中のみ表示。ユーザ手入力時は非表示 |
| ローカル作成先パス | 文字列(絶対パス) | 必須 | 最終作成先ディレクトリパス。初期値は `/Users/.../gitspace/{projectName}` |
| テンプレート | 文字列 | 省略可 | `"default"` 固定（UI 上は非表示・API 送信時に自動設定） |

## 3. 主要アクション

1. 画面表示時に自動で機能抽出・候補生成
- `POST /api/v1/projects/extract-features`
- `POST /api/v1/projects/suggest-name`

2. 「概要入力に戻る」押下
- `/overview?overviewId=...` へ遷移し、対象概要本文を再表示

3. 「初期ディレクトリ作成」押下
- `POST /api/v1/projects/init-directory`

## 4. 画面状態

| 変数 | 型 | 説明 |
| --- | --- | --- |
| `overviewId` | 文字列 | URL クエリから取得した概要ID |
| `features` | 文字列[] | 抽出された機能リスト |
| `components` | 文字列[] | 抽出された構成要素リスト |
| `candidates` | 文字列[] | プロジェクト名候補 |
| `candidateItems` | 候補オブジェクト[] | 候補名・AI提案フラグ・由来説明 |
| `selectedProjectName` | 文字列 | 選択または手入力したプロジェクト名 |
| `selectedCandidate` | 候補オブジェクト | 現在選択中の候補。AI候補由来表示に利用 |
| `localPath` | 文字列 | 最終作成先パス |
| `localPathEdited` | 真偽値 | ユーザが手動でパス編集済みか |
| `hasAutoExtracted` | 真偽値 | 自動抽出の重複実行防止フラグ |
| `directoryStatus` | 文字列 | 初期ディレクトリ作成結果（"success" 等） |
| `errorMessage` | 文字列 | エラーメッセージ（空の場合は非表示） |

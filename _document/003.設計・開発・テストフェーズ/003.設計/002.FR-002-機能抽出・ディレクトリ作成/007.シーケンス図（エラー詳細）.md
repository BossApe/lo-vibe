## 2. overviewId が存在しない（404）

**E-01**: extract-features または suggest-name で存在しない overviewId を指定

```mermaid
sequenceDiagram
    actor User as ユーザ
    participant FE as フロントエンド<br>(SCR-002)
    participant API as musuhi-api<br>(Handler)
    participant SVC as Service 層
    participant REPO as Repository 層

    User->>FE: /projects/setup?overviewId={missing-uuid} に遷移
    FE->>FE: onMount → overviewId セット
    FE->>FE: $effect → handleExtract() 自動実行
    FE->>API: POST /api/v1/projects/extract-features<br>{"overviewId": "{missing-uuid}"}
    activate API
    API->>SVC: ExtractFeatures(overviewID)
    activate SVC
    SVC->>REPO: FindByID(uuid)
    REPO-->>SVC: pgx.ErrNoRows
    SVC-->>API: ErrNotFound
    deactivate SVC
    API-->>FE: HTTP 404<br>{"error":{"code":"NOT_FOUND","message":"..."}}
    deactivate API
    FE->>FE: errorMessage = "..." (API の message)
    FE-->>User: エラーメッセージ表示
```

## 5. プロジェクト名バリデーションエラー（422）

**E-04**: `projectName` がパターン `^[a-zA-Z0-9][a-zA-Z0-9_-]*$` に違反

```mermaid
sequenceDiagram
    actor User as ユーザ
    participant FE as フロントエンド<br>(SCR-002)
    participant API as musuhi-api<br>(Handler)
    participant SVC as Service 層

    User->>FE: "bad name!" を入力して「初期ディレクトリ作成」クリック
    FE->>API: POST /api/v1/projects/init-directory<br>{"projectName":"bad name!","localPath":"/...","template":"default"}
    activate API
    API->>SVC: InitDirectory("bad name!", ...)
    activate SVC
    SVC->>SVC: projectNamePattern.MatchString → false<br>(ErrValidation)
    SVC-->>API: ErrValidation
    deactivate SVC
    API-->>FE: HTTP 422<br>{"error":{"code":"VALIDATION_ERROR","message":"..."}}
    deactivate API
    FE->>FE: errorMessage = "..."
    FE-->>User: エラーメッセージ表示
```

## 7. ディレクトリ作成失敗（500）

**E-06**: OS レベルでのディレクトリ作成失敗（権限不足など）

```mermaid
sequenceDiagram
    actor User as ユーザ
    participant FE as フロントエンド<br>(SCR-002)
    participant API as musuhi-api<br>(Handler)
    participant SVC as Service 層
    participant OS as OS ファイルシステム

    User->>FE: 「初期ディレクトリ作成」クリック
    FE->>API: POST /api/v1/projects/init-directory<br>{"projectName":"demo","localPath":"/root/no-perm","template":"default"}
    activate API
    API->>SVC: InitDirectory("demo", "/root/no-perm", "default")
    activate SVC
    SVC->>OS: os.MkdirAll("/root/no-perm", 0755)
    OS-->>SVC: permission denied
    SVC-->>API: fmt.Errorf("projectService.InitDirectory: %w", err)
    deactivate SVC
    API-->>FE: HTTP 500<br>{"error":{"code":"INTERNAL_ERROR","message":"サーバーエラーが発生しました"}}
    deactivate API
    FE->>FE: errorMessage = "初期ディレクトリ作成に失敗しました。"
    FE-->>User: エラーメッセージ表示
```

## 1. エラー一覧

| # | エラー種別 | HTTP | code |
| --- | --- | --- | --- |
| E-01 | owner 空 | 422 | `VALIDATION_ERROR` |
| E-02 | repoName 重複 | 422 | `VALIDATION_ERROR` |
| E-03 | Forgejo 認証失敗 | 500 | `INTERNAL_ERROR` |
| E-04 | git 実行失敗 | 500 | `INTERNAL_ERROR` |

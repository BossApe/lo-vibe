# FR-003-画面設計書（SCR-003）

前: [FR-003-API仕様書](FR-003-API仕様書.md) | [一覧](README.md) | 次: [FR-003-テスト仕様書](FR-003-テスト仕様書.md)

## 1. 画面概要

- 画面ID: SCR-003
- 画面名: Forgejoリポジトリ作成確認
- URL: `/projects/{id}/repository`
- 目的: Forgejoリポジトリ作成と initial commit/push 実行

## 2. 入力項目

| 項目 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| owner | 文字列 | 必須 | Forgejo owner（ユーザまたは組織） |
| repoName | 文字列 | 必須 | 作成するリポジトリ名 |
| visibility | Enum | 必須 | `public` / `private` |
| localPath | 文字列（絶対パス） | 必須 | push 対象ローカルディレクトリ |
| commitMessage | 文字列 | 必須 | 初回 commit メッセージ |

## 3. 主要アクション

1. 「リポジトリ作成と push」押下
- `POST /api/v1/projects/with-external`

2. 成功時
- `repositoryUrl` を表示
- `pushStatus=success` を表示
- SCR-004 への遷移導線を表示

3. 失敗時
- `error.message` を表示
- 再実行ボタンを表示

## 4. 画面状態

| 変数 | 型 | 説明 |
| --- | --- | --- |
| owner | 文字列 | owner 入力値 |
| repoName | 文字列 | リポジトリ名入力値 |
| visibility | 文字列 | `public/private` |
| localPath | 文字列 | ローカル作成先パス |
| commitMessage | 文字列 | 初回コミットメッセージ |
| repositoryUrl | 文字列 | 成功時の作成先URL |
| externalProjectId | 文字列 | 成功時のForgejoリポジトリID |
| pushStatus | 文字列 | `pending/success/failed` |
| isLoading | 真偽値 | API 呼び出し中フラグ |
| errorMessage | 文字列 | エラーメッセージ |

# FR-002-画面設計書（SCR-002）

前: [FR-002-API仕様書](FR-002-API仕様書.md) | [一覧](README.md) | 次: なし

## 1. 画面概要

- 画面ID: SCR-002
- 画面名: プロジェクト名・構成要素確認
- URL: `/projects/setup`
- 目的: 機能抽出結果確認、プロジェクト名確定、初期ディレクトリ作成

## 2. 入力項目

| 項目 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| 概要ID | 文字列(UUID) | 条件付き | FR-001 保存結果ID。URL クエリパラメータ `overviewId` から取得し、画面には表示しない。未指定時は抽出処理を実行しない |
| モデル運用モード | 文字列 | 条件付き | `fast` / `balanced` / `quality`。`GET /api/v1/projects/name-suggestion-profile` の `enabled=true` の場合のみ表示 |
| プロジェクト名 | 文字列 | 必須 | ディレクトリ名・Forgejoプロジェクト名として利用可能な文字のみ（`^[a-zA-Z0-9][a-zA-Z0-9_-]*$`）。既存プロジェクト名は選択不可（重複禁止）。候補配列の先頭にはAI神様候補を最低1件含む。|
| プロジェクト名由来説明 | 文字列 | 条件付き | 候補選択中に表示。`items.reason` がある場合はその文言、ない場合はフォールバック説明を表示。ユーザ手入力時は非表示 |

※「ローカル作成先パス」入力項目は廃止。初期ディレクトリ作成先は musuhi-apiコンテナ内 `/app/project/{プロジェクト名}` に固定。

## 3. 主要アクション

1. 画面表示時に自動で機能抽出・候補生成
- `POST /api/v1/projects/extract-features`
- `POST /api/v1/projects/suggest-name`
- `GET /api/v1/projects/name-suggestion-profile`（モード切替可否・現在値を取得）

2. モデル運用モード変更
- `PUT /api/v1/projects/name-suggestion-profile`
- 更新成功後、再度 `POST /api/v1/projects/extract-features` と `POST /api/v1/projects/suggest-name` を実行

3. 「概要入力に戻る」押下
- `/overview?overviewId=...` へ遷移し、対象概要本文を再表示

4. 「初期ディレクトリ作成」押下
- `POST /api/v1/projects/init-directory`

## 4. 画面状態

| 変数 | 型 | 説明 |
| --- | --- | --- |
| `overviewId` | 文字列 | URL クエリから取得した概要ID |
| `features` | 文字列[] | 抽出された機能リスト |
| `components` | 文字列[] | 抽出された構成要素リスト |
| `candidates` | 文字列[] | プロジェクト名候補 |
| `candidateItems` | 候補オブジェクト[] | 候補名・AI提案フラグ・由来説明 |
| `selectedProjectName` | 文字列 | 選択または手入力したプロジェクト名 |
| `selectedCandidate` | 候補オブジェクト | 現在選択中の候補 |
| `selectedCandidateReason` | 文字列 | 表示用説明文。`selectedCandidate.reason` 優先、未設定時はフォールバック文言 |
| `localPath` | 文字列 | 最終作成先パス |
| `localPathEdited` | 真偽値 | ユーザが手動でパス編集済みか |
| `hasAutoExtracted` | 真偽値 | 自動抽出の重複実行防止フラグ |
| `directoryStatus` | 文字列 | 初期ディレクトリ作成結果（"success" 等） |
| `errorMessage` | 文字列 | エラーメッセージ（空の場合は非表示） |
| `isLoading` | 真偽値 | API 呼び出し中フラグ |
| `isSwitchingProfile` | 真偽値 | モデル運用モード切替中フラグ |
| `profileEnabled` | 真偽値 | モデル運用モード切替 UI を表示するか |
| `modelProfile` | 文字列 | 現在選択中のモデル運用モード |
| `availableProfiles` | 文字列[] | 選択可能なモデル運用モード一覧 |

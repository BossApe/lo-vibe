= tools ディレクトリ

プロジェクトをスムーズに進めるためのツールスクリプトを格納するディレクトリです。

== 概要

開発・運用作業の効率化や自動化を目的としたスクリプト類を管理します。

== ディレクトリ構成

[source]
----
tools/
├── README.adoc       # このファイル
└── (ツールスクリプト群)
----

== スクリプト一覧

[cols="2,1,3", options="header"]
|===
|ファイル名
|種別
|概要

|(追加予定)
|
|

|===

== 利用方法

各スクリプトのヘルプオプション (`-h` / `--help`) または個別のコメントを参照してください。

== 規約

* スクリプトファイル名はケバブケース（例: `setup-env.sh`）とする
* ファイル先頭にスクリプトの概要・使用方法をコメントで記載する
* 実行権限を付与する (`chmod +x <script>`)
* 新規スクリプトを追加した場合は本 README のスクリプト一覧を更新する

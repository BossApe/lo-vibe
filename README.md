# Musuhi

## 概要

Musuhi は、バイブコーディングをローカル環境で実行するためのツールです。本リポジトリでは Musuhi 自体の開発をCopilotと共に行いますが、MusuhiのVersion 1.0が完了してからはMusuhiを活用してMusuhi自身の開発を行う予定です。

- バイブコーディングとは<br>
AI を活用してソフトウェア開発のプロセスを効率化する手法であり、プロジェクト管理、要件定義、設計、コーディング、テストなどの各フェーズで AI を活用します。
- Musuhi の名前について<br>
「産霊（むすひ）」とは、神道においてモノを創造する力を指す言葉です。
